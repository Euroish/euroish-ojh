# AGENTS (optional)

Treat these as **work modes**, not personas.

## Explore
Read-only inspection. Find context, boundaries, unknowns.

## Plan
Turn findings into a narrow plan, model, or decision.

## Execute
Implement the smallest useful change.

## Review
Check regressions, edge cases, and whether the output matches the task.

## Rule
Do not force all four modes every time.
Use only the minimum needed for the task.
